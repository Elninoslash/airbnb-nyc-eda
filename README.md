# Exploratory Data Analysis of Airbnb NYC 2019

## Project Overview
This project performs Exploratory Data Analysis (EDA) on the Airbnb NYC 2019 dataset to understand pricing behavior, availability patterns, room type distribution, and customer demand.

## Dataset
- Source: Airbnb NYC 2019
- Records: ~48,895 listings
- Features: Pricing, location, room type, availability, and reviews

## Key Objectives
- Analyze pricing patterns across room types and locations
- Study availability and booking behavior
- Identify factors influencing customer engagement

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Google Colab
